Step 1: Upload Widget Files

Upload these files to your WordPress site:

  • metalogics-chatbot.iife.js (from your dist folder)
  • metalogics-chatbot.css (from your dist folder)

Upload location: /wp-content/uploads/chatbot/

Step 2: Add Code to WordPress

Add this code to your WordPress site before the closing </body> tag:

<!-- Retell SDK (for voice features) --> <script src="https://cdn.jsdelivr.net/npm/retell-client-js-sdk@2.0.0/dist/retell-client-js-sdk.min.js"></script> <!-- Metalogics Widget CSS --> <link rel="stylesheet" href="https://yoursite.com/wp-content/uploads/chatbot/metalogics-chatbot.css" /> <!-- Metalogics Widget Script --> <script src="https://yoursite.com/wp-content/uploads/chatbot/metalogics-chatbot.iife.js" data-api-key="YOUR_GEMINI_API_KEY" data-api-url="https://metabot-ai-backend-production.up.railway.app" data-retell-agent-id="YOUR_RETELL_AGENT_ID" data-brand-color="#3b82f6" data-position="bottom-right" ></script>

Step 3: Configure Your Settings

Replace these placeholders with your actual values:

  • YOUR_GEMINI_API_KEY - Your Google Gemini API key
  • YOUR_RETELL_AGENT_ID - Your Retell agent ID (optional for voice)
  • https://yoursite.com - Your actual WordPress domain
